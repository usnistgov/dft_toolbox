============
Installation
============


Stable release
--------------

To install DFT Toolbox, run this command in your terminal:

.. code-block:: console

    $ pip install dft_toolbox

This is the preferred method to install DFT Toolbox, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for DFT Toolbox can be downloaded from the `GitLab repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://gitlab.com/hnr2/dft_toolbox

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://git@gitlab.nist.gov/hnr2/dft-toolbox/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _GitLab repo: https://git@gitlab.nist.gov/hnr2/dft-toolbox
.. _tarball: https://git@gitlab.nist.gov/hnr2/dft-toolbox/tarball/master
